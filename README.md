# homebridge-picamera-lightsensor

This is a [homebridge](https://github.com/nfarina/homebridge) plugin which uses the raspberry-pi camera to emulate a ambientlight sensor

# Configuration

 ```
"accessories": [
	{
		"accessory": "PiCameraLightsensor",
		"name": "LightSensor",
		"poll_interval": 300
	}
]
 ```
